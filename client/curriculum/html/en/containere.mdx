  And finally we reach the <FormattedText as="strong"> {'<div>'} </FormattedText> container, the most popular element of this type, which unfortunately is incorrectly used a lot of times.

  <FormattedText as="strong"> {'<div>'} </FormattedText> is a generic container, without any special meaning. While elements such as <FormattedText> {'<nav>'} </FormattedText> suggest navigation links, and <FormattedText> {'<main>'} </FormattedText> represents the most important part of the page, <FormattedText> {'<div>'} </FormattedText> has no special meaning thus it should be used only when <span className="text-bold"> no other container element makes sense </span>.

  For example, in the <a href="https://unsplash.com/">Unsplash </a> image gallery, the <FormattedText> {'<div>'} </FormattedText> element is perfect as a container for those elements, as they don't hold any special meaning. We just need to position them on the edges of the image.
