  HTML offers six elements for headings: <FormattedText as="strong">{'<h1>'}</FormattedText>, <FormattedText as="strong">{'<h2>'}</FormattedText>, <FormattedText as="strong">{'<h3>'}</FormattedText>, <FormattedText as="strong">{'<h4>'}</FormattedText>, <FormattedText as="strong">{'<h5>'}</FormattedText> and <FormattedText as="strong">{'<h6>'}</FormattedText>.
  
  <FormattedText as="strong">{'<h1>'}</FormattedText> is used for the main title of the page. The rest of the headings are used for sub-titles, in order of importance. It's recommended to have <strong>just one {'<h1>'}</strong> in each page, to help search engines figure out the subject discussed there.

  <Highlight className="my-5" language="html" code={`<h1> Texts | HTML Lesson </h1>`} />

  The browsers add their own initial styling to HTML elements, thus <FormattedText as="strong">{'<h1>'}</FormattedText> will have a bigger font size, while <FormattedText as="strong">{'<h6>'}</FormattedText> a much smaller one. However, this rule is not set in stone.

  We have total control over those styles via <Link prefetch={false} href="/css/intro">CSS</Link>, which we'll discuss in a follow up lesson.

  In ordinary language we often underline words to change the meaning of a sentence, and we want to mark certain words as important or different in some way.
  
  HTML offers different semantic elements that allow us to give special meaning to different words/phrases. One of those elements is the <FormattedText as="strong">{'<em>'}</FormattedText> tag.

  <Highlight
    className="my-5"
    language="html"
    code={`<p>
  I am <em> very happy </em> you got the job!
</p>`} />

  We also have access to the <FormattedText as="strong">{'<strong>'}</FormattedText> element, used to mark an item as very important. As a semantic element, it is recognized by screen readers and played with a different tone of voice.

  Although browsers show this content in a bold font, <strong>we shouldn't use</strong> the <FormattedText as="span">{'<strong>'}</FormattedText> element only to get this styling result.
  
  So, if you have a normal word/phrase but you want to mark it as bold, we recommend using the <FormattedText as="span">{'<span>'}</FormattedText> element and CSS rules for this.
